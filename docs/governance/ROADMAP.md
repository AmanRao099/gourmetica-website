# Roadmap

## Phase 1: Foundation (Current)
- Establish Governance, Architecture, and Design "Bible".
- Define ADRs and strict boundaries.

## Phase 2: Core Engines
- Theme, Tokens, Motion, and Accessibility libraries.
- Platform wrappers (Analytics, SEO).

## Phase 3: Core UI
- Build the atomic layer (Primitives, Typography, Components).

## Phase 4: Assembly
- Build generic `blocks/` (Hero, CTA, LogoCloud).

## Phase 5: Business Logic
- Implement Feature domains and CMS Repositories.
- Assemble Next.js routes.

## Phase 6: Polish
- Execute Performance Budgets.
- E2E Testing.
