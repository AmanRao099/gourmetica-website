# Visual Language

This document dictates the rules for typography, whitespace, and iconography in the Gourmetica Platform.

## 1. Typography Rules
Typography is our primary design element. It must be oversized, strictly hierarchical, and impeccably spaced.

### Hierarchy & Weights
- Never use more than three font weights in a single section.
- **Font:** Mont (Primary Brand Font).
- Large text requires tight line-heights (e.g., 1.1 or 1.05).
- Body text requires generous line-heights (e.g., 1.6 or 1.65).

### Example
**✅ Correct (Strong, tight, readable)**
```text
Growth Marketing
That Delivers
Results.
```

**❌ Incorrect (Awkward line breaks, loose leading)**
```text
Growth
Marketing

That
Delivers

Results.
```

## 2. Whitespace Rules
Whitespace is a deliberate design element, not just empty space.

### The 8-Point System
All spacing must adhere to the 8-point system (4, 8, 12, 16, 20, 24, 32, 40, 48, 56, 64, 72, 80, 96, 120, 160). No magic numbers allowed.

### Spacing Contexts
- **Component internal spacing:** Tight (e.g., 8px - 24px) to show relationship.
- **Section spacing:** Generous (e.g., 96px, 120px) to pace the user's reading.

## 3. Iconography
- **Library:** Lucide React only.
- **Style:** Stroke icons only. Absolutely no filled icons.
- **Consistency:** Consistent stroke width (usually 2px or 1.5px) and size.
- **Prohibited:** No emojis. No mismatched icon libraries.
