# Motion Philosophy

Motion should support the content and enhance the storytelling, never distract from it. Luxury websites rely on calm, supportive motion that highlights the craftsmanship of the design.

## Core Motion Guidelines

1. **Support Content**: Every animation must serve a clear purpose—guiding the reader's eye, indicating loading state transitions, or supporting editorial hierarchy.
2. **Entrances Limit**: The maximum duration for any entrance or fade transition is **800ms**.
3. **Hover Animations**: Interactive states and button hovers must remain swift and responsive, keeping durations between **150ms and 250ms**.
4. **Scroll Reveals**: Animations triggered by scroll entry must play **once only** to avoid continuous distraction.
5. **No Infinite Animations**: Infinite loops or repeating decorative animations are strictly prohibited, with the exception of intentional branded marquee streams.
6. **Easing Curves**:
   - Use standard **ease-out** curves for layout entrances (calm and natural arrival).
   - Use **ease-in-out** curves for user interactive states.
