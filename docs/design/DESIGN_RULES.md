# Luxury Design Rules & Principles

## Luxury Principles
- **Never center long paragraphs**: Maintain left-aligned block text for legibility and structure.
- **Never use more than one primary CTA per section**: Primary actions must stand alone, secondary choices should be outlined or text links.
- **Every section begins with whitespace, not borders**: Borders are structural, not decorative. Leverage whitespace to divide sections.
- **Images always breathe more than text**: Ensure plenty of margin/padding around visuals.
- **Motion guides attention, never requests it**: Animations must be subtle, purposeful, and quiet (e.g. clip reveals, slow zoom transitions).
- **Typography is the primary design element**: Set typographic rhythm and scale carefully; never use mismatched fonts.
- **Color is used for emphasis, not decoration**: Keep base tones neutral and apply brand red selectively.
- **Components should feel calm before they feel interactive**: Avoid busy UI components.
- **Every hover state should be subtle enough that it doesn't distract from reading**: Micro-animations must have extremely light, elegant transitions.
- **If removing an element improves the page, remove it**: Simplicity is the ultimate luxury.
