# Content System

## 1. Copywriting Rules
Content dictates design. Never write content to fit a box; build the box to fit the optimal content.

### Formatting
- **Heading length:** Keep headings punchy. 3-6 words is ideal for Display sizes.
- **Body length:** Never exceed 70 characters per line for readability.
- **Sentence Case:** Use Sentence case for all headings, subheadings, and buttons. Never use ALL CAPS or Title Case unless it is an Eyebrow or Overline.
- **Capitalization:** Gourmetica, not gourmetica or GOURMETICA.

### Tone
- **Hospitality Tone:** Gracious, attentive, high-touch.
- **Marketing Tone:** Persuasive, data-driven, results-oriented, yet refined.

## 2. Content Templates

### Hero Block
- **Eyebrow:** (Optional) 1-3 words setting context.
- **Heading:** High impact, clear value proposition.
- **Subheading:** 1-2 sentences expanding on the heading.
- **Primary CTA:** Action-oriented (e.g., "View case studies").
- **Secondary CTA:** (Optional) Lower friction (e.g., "Learn more").

### Section Header
- **Eyebrow:** Categorical.
- **Heading:** The main takeaway of the section.
- **Description:** Brief elaboration.

### Call To Action (CTA)
- Button wording must complete the sentence: "I want to..." (e.g., "...Grow my reservations" -> "Grow my reservations").
