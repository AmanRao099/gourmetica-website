# Color Tokens

Our color tokens establish the premium, confident, and editorial aesthetic of Gourmetica. We use a limited, highly intentional palette.

## Base Tokens
- **Primary:** `#E42528` - Used exclusively for the most critical actions and brand moments. Do not overuse.
- **Black:** `#000000` - Our primary heavy background and highest contrast text color.
- **White:** `#FFFFFF` - Our primary light background and highest contrast text color for dark mode.

## Neutral Scale (50 to 950)
Rather than standard gray, our neutrals are slightly tinted to maintain a luxury, editorial feel. 

| Token | Value | Intent |
| :--- | :--- | :--- |
| `neutral-50` | `#FAFAFA` | Page backgrounds (Light mode) |
| `neutral-100` | `#F5F5F5` | Secondary backgrounds, subtle fills |
| `neutral-200` | `#E5E5E5` | Borders, subtle dividers |
| `neutral-300` | `#D4D4D4` | Disabled states, heavy borders |
| `neutral-400` | `#A3A3A3` | Placeholder text |
| `neutral-500` | `#737373` | Secondary text (Dark mode) |
| `neutral-600` | `#525252` | Secondary text (Light mode) |
| `neutral-700` | `#404040` | Hover states on dark |
| `neutral-800` | `#262626` | Card backgrounds (Dark mode) |
| `neutral-900` | `#171717` | Subtle backgrounds (Dark mode) |
| `neutral-950` | `#0A0A0A` | Page backgrounds (Dark mode) |

## Semantic Tokens
Used only for explicit user feedback. Never for branding or decoration.
- **Success:** `#10B981` (Emerald)
- **Warning:** `#F59E0B` (Amber)
- **Error:** `#EF4444` (Red)
