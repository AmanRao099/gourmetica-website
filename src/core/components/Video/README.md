# Video

The `Video` component standardizes `<video>` elements with consistent styling and aspect ratios.

## Props
- `radius`: `none`, `sm`, `md` (default), `lg`, `full`.
- `aspectRatio`: `auto`, `video` (default), `square`.
- Inherits all standard `<video>` attributes (e.g. `controls`, `poster`, `autoPlay`).
