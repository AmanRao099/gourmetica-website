# Radio

The `RadioGroup` and `RadioGroupItem` components are built on top of `@radix-ui/react-radio-group` for WAI-ARIA compliance.

## Usage
Wrap `RadioGroupItem`s inside a `RadioGroup` and always pair them with `<Label htmlFor="...">`.

## Props
- `hasError`: Highlights the radio button in red.
- Supports all Radix UI Radio Group props.
