# Avatar

The `Avatar` component displays a user profile image or a fallback initial. Built on top of `@radix-ui/react-avatar`.

## Composition
Use it by composing its parts:
```tsx
<Avatar size="lg">
  <AvatarImage src="image.jpg" alt="Jane Doe" />
  <AvatarFallback>JD</AvatarFallback>
</Avatar>
```
