# Icon

The `Icon` component is a strict wrapper around `lucide-react` icons. It enforces standard sizes.

## Props
- `icon`: The lucide-react icon component to render.
- `size`: `sm` (16px), `md` (20px, default), `lg` (24px), `xl` (32px).
- `className`: Use tailwind text colors (`text-brand-500`) to color the icon.
