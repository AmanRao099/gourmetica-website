# Text

The `Text` component renders long-form body text. It enforces the Raleway font family.

## Props
- `size`: `body-xl`, `body-lg`, `body` (default), `body-sm`.
- `weight`: `light`, `normal` (default), `medium`, `semibold`.
- `align`: `left` (default), `center`, `right`, `justify`.
- `as`: Overrides the rendered HTML tag (e.g. `as="span"`). Defaults to `p`.
