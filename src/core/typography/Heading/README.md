# Heading

The `Heading` component renders all primary and secondary titles. It enforces the Mont font family.

## Props
- `size`: `display-xl`, `display-lg`, `heading-xl`, `heading-lg`, `heading-md`, `heading-sm`.
- `weight`: `normal`, `medium`, `semibold`, `bold` (default), `black`.
- `level`: The semantic HTML heading level (`1` through `6`). Defaults to `2`.
- `as`: Overrides the rendered HTML tag without affecting semantics if necessary (e.g. `as="div"`).
