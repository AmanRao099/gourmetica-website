# FeatureGridBlock

The `FeatureGridBlock` displays a grid of features or services, optionally with icons, titles, and descriptions.

## Props
- `title` (string): Optional headline
- `subtitle` (string): Optional eyebrow
- `description` (string): Optional paragraph
- `features` (Array): Array of `{ title, description, icon?, href? }`
- `columns` (2 | 3 | 4): Grid columns on desktop
- `theme` (`light` | `dark`)
- `variant` (`minimal` | `cards`)
