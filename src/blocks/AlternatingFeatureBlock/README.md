# AlternatingFeatureBlock

The `AlternatingFeatureBlock` displays a list of features with alternating image and text layouts.

## Props
- `title` (string): Optional headline
- `subtitle` (string): Optional eyebrow
- `description` (string): Optional paragraph
- `features` (Array): Array of `{ title, subtitle?, description, media, actions? }`
- `theme` (`light` | `dark`)
