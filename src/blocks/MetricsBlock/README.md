# MetricsBlock

The `MetricsBlock` displays a grid of large numbers and labels to highlight key statistics.

## Props
- `title` (string): Optional headline
- `subtitle` (string): Optional eyebrow
- `description` (string): Optional paragraph
- `metrics` (Array): Array of `{ label, value, description? }`
- `theme` (`light` | `dark` | `brand`)
- `columns` (2 | 3 | 4): Grid columns on desktop
