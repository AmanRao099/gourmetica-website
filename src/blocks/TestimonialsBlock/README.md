# TestimonialsBlock

The `TestimonialsBlock` displays quotes from users or customers to build trust and social proof.

## Props
- `title` (string): Optional headline
- `subtitle` (string): Optional eyebrow
- `description` (string): Optional paragraph
- `testimonials` (Array): Array of `{ quote, author: { name, role, company, avatar } }`
- `layout` (`grid` | `featured`)
- `theme` (`light` | `dark`)
