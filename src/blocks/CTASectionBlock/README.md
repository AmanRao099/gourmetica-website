# CTASectionBlock

The `CTASectionBlock` provides a visually striking call-to-action area intended to drive user conversion.

## Props
- `title` (string): Primary headline
- `subtitle` (string): Optional eyebrow above headline
- `description` (string): Optional paragraph
- `actions` (ReactNode): Buttons or CTAs
- `theme` (`light` | `dark` | `brand`): The background color theme
- `variant` (`centered` | `split` | `banner`): The visual layout template
